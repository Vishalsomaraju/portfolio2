# Design System Rules (STRICT)

This project follows a premium, minimal design system.

## Core Principle
- 90% neutral
- 8% accent
- 2% glow

## Accent Rules
Accent color: `#E07A5F`

**Allowed usage:**
- CTA buttons
- Hover states
- Cursor interactions
- Focus rings

**Forbidden:**
- Large backgrounds
- Paragraph text
- Over-highlighting UI

## Light Mode
- Soft, warm, breathable
- No harsh contrast

## Dark Mode
- Deep layered backgrounds
- No flat blacks
- Use elevation through subtle contrast

## Typography
- Display → strong, minimal
- Body → clean and readable
- Avoid excessive sizes

## Spacing
- Follow strict spacing scale
- Prefer whitespace over clutter

## Visual Philosophy
- Minimal but expressive
- Motion replaces decoration
- Every element must justify its presence

Violation of these rules = bad output.
