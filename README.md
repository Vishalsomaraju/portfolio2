# Interactive 3D Portfolio

A premium, award-level portfolio website following the "From Chaos → Systems" visual narrative. 
Built using a Modular Monolith architecture geared entirely for high-performance interactivity and scalability.

## Architecture

- **Next.js 14 App Router** for routing and SSG.
- **Framer Motion + GSAP + Lenis** for scroll-synced timelines.
- **React Three Fiber + Drei** for high-performance Canvas interactions.
- **Zustand** for system synchronization between DOM and WebGL.

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Open [http://localhost:3000](http://localhost:3000) with your browser.

## Custom CLI Commands

View `ai/.claude/commands` to see available scaffolding shortcuts to generate additional motion sections.
